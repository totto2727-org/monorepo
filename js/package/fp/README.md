# @totto2727/fp

A TypeScript functional programming utility library that provides unified re-exports and bridge modules for Effect, option-t, and other FP libraries.

## Installation

This package is available on [NPM](https://npmx.dev/package/@totto2727/fp) and [JSR](https://jsr.io/@totto2727/fp).

```bash
npm install @totto2727/fp
yarn add @totto2727/fp
pnpm add @totto2727/fp
bun add @totto2727/fp
vlt install @totto2727/fp

# Deno (via JSR)
deno add jsr:@totto2727/fp
```
