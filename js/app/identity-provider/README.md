# identity-provider

`identity-provider` is the local Identity Provider for the feed platform. It provides Magic Link, Passkey, OAuth 2.0, and OpenID Connect flows through Better Auth.

## Documentation

- [Authentication flow and OAuth/OIDC status](docs/auth-flow.md)

The linked document uses `feed-platform-web` and `feed-platform-backend` as the worked example for local authorization-code login, token exchange, backend JWT verification, and known security/compliance gaps.
