# Shared Artifact Index

Each authoring guide in references/ corresponds to a same-named skeleton in templates/.

- Milestone: [guide](references/milestone.md) and [template](templates/milestone.md).

## Usage

1. Select the artifact pair.
2. Read the guide before copying the template.
3. Fill every placeholder or mark it explicitly as not applicable.
4. Review the result against the guide's quality criteria.
