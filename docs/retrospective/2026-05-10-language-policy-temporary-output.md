# Retrospective: 2026-05-10-language-policy-temporary-output

- **Identifier:** 2026-05-10-language-policy-temporary-output
- **Created at:** 2026-05-10

## Context

During the `html-doc` and temporary `geo/2d` explanation work, the language boundary was clarified.

## Reusable insight

- Use English by default for repository work.
- Use Japanese only for user-facing temporary outputs that are not final artifacts.
- Decide whether an output is temporary or final before choosing the language.
- Temporary exploratory HTML may be Japanese when it improves short-term readability.
- Long-lived repository instructions, skills, and documentation should remain English by default.

## Scope note

The temporary `geo-mbt` explanation files are planned for deletion, so this retrospective does not track them as translation work.
